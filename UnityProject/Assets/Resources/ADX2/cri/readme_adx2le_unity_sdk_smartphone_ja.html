はじめにお読みください - CRI ADX2 LE SDK for Unity

１．　目的別の導入ガイド


　本SDKにはいくつかのドキュメントが収録されています。ユーザーの目的別に最初にどこを読めば良いかを紹介します。

（１）　ADX2でどんな事が出来るか知りたい人

　WEB公開しているプラグインマニュアル内の「CRIWARE Unity Plug-in の紹介」をご覧ください。
　このマニュアルには ADX2LE版 CRIWARE Unity Plugin には含まれない機能 (主にムービ再生機能) に関する記述も含みます。

（２）　ADX2をプログラムに組み込みたい人

　プラグインマニュアルの「CRIWARE Unity Plug-inの使い方」をご覧ください。
　具体的な使用方法は「CRI Atomコンポーネント」や「CRI ADX2サンプル」をご覧ください。

（３）　ADX2でサウンドデータを作ってみたい人

　ツールの使用方法詳細はツールマニュアルをご覧ください。

２．　SDKドキュメント


（１）　変更履歴

  CRI ADX2 LE SDK for Unity

  CRI ADX2 LE Unity Plug-in
  プラグイン変更履歴 プラグインの詳細な変更履歴です。
