１．プラグインバージョン
  - CRIWARE Unity   Ver.2.40.22LE

２．動作環境
本バージョンの動作要件および動作確認環境は以下になります。

２．１ 動作要件
本プラグインを動作させるには、Unityの動作要件に加えて
以下の要件を満たす必要があります。

(1) Unity
  - Unity 2019以降

(2) PC
  - Windows 8.1 / 10
  - DirectX エンドユーザーランタイム (June 2010)
  - Microsoft .NET Framework 3.5 SP1
  - Microsoft .NET Framework 4.0

(3) Mac
  - Mac OS X 10.11以降 (64bitのみ)

(4) Android
  - ARMv7/ARM64/x86対応のCPU、Android OS 2.3以上を搭載した機種
  ※エミュレータ上での動作確認はしておりません。

(5) iOS
  - armv7/arm64対応のCPU、iOS10以降がインストールされた端末
  ※シミュレータ上では動作しません。
  （Unityのネイティブプラグインの動作制限のため）
